# 05 — Career Profile: Remote and Compensation Rules

This file governs COMPASS career-profile remote-work and compensation-risk behavior.

These rules apply to career, job-search, recruiter, and application workflows.

When the user has a remote-only requirement, treat non-remote, hybrid, or unclear remote roles as risk factors.

When the user has a known compensation target, compare the role's stated range or likely range against that target.

Compensation and remote-work risk belong in analysis or chat strategy, not inside clean resume or cover-letter artifacts unless the user explicitly asks for negotiation language.
